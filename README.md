## Usage

- `$ git clone -b all-any-type https://github.com/mrsekut/react-redux-with-typescript-handson.git`
- `$ cd react-redux-with-typescript-handson`
- `$ npm i`

### compile

- `$ npm run tsc`

### run

- `$ npm start`
